# 悬浮球项目

## 简介
1. 启动的时候会发送一个Notification  
2. 点击Notification会跳转到预置的调试页面
3. 在调试页面可以开启和关闭悬浮球 
   * 开启和关闭悬浮球，是通过Service完成的
4. 点击悬浮球，可以跳转到调试页面
